Deathmatch

server-authoritative FPS netcode — web demo

Solo

Play against a training dummy. The client talks to an in-process authority over a loopback socket, so all the netcode — prediction, reconciliation, interpolation, lag compensation — runs locally. Dial the emulated link with the number keys and watch the netgraph react.

Multiplayer WIP

Player-vs-player against a dedicated server. The browser has no raw UDP, so this needs a WebTransport (QUIC) transport plus a server that speaks it — not wired up yet. Native builds already do PvP over UDP (--connect).
