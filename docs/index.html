2D + 3D in one engine Cross-platform (Windows / Linux / macOS / iOS / Android / Web) PBR forward renderer with out-of-the-box post-processing effects Animation system (skeletal animation WIP) Physics engine integration (Box2D / Bullet) UI framework integration (RmlUi) Networking toolkit with client-side prediction, server reconciliation, and lag compensation Built-in job system Lua scripting support
