Getting Started

Build Atmospheric from source and run your first example.

🚧

This page is a placeholder. Full getting-started documentation is coming soon.
In the meantime, see the README on GitHub for build instructions.

Prerequisites

  • CMake 3.20+ — cmake.org
  • A C++17 compiler — Clang (macOS), GCC (Linux), MSVC (Windows)
  • EMSDK (optional, for WebAssembly builds) — emscripten.org

Clone

git clone --recurse-submodules https://github.com/painfulexistence/Atmospheric.git
cd Atmospheric

Setup Vcpkg

./vcpkg/bootstrap-vcpkg.sh -disableMetrics

Native Build

cmake --preset=dev
cmake --build --preset=dev

WebAssembly Build

# WebGL 2.0 (stable)
./scripts/buildWasm.sh release
